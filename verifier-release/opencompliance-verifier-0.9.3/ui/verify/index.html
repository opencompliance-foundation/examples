Local UX

Verify Workbench

This is a local browser front end for the deterministic Verify API. Serve this repository over `http://127.0.0.1`, run the API on `127.0.0.1:8788`, and you can load a synthetic ExampleCo bundle or submit a fixture-path request without leaving the browser.

cd projects/dev/opencompliance
python3 scripts/serve_verify_api.py --port 8788

# second terminal, same directory
python3 -m http.server 8000

# then open http://127.0.0.1:8000/ui/verify/
