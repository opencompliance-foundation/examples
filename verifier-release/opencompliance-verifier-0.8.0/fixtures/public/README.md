# Public Fixtures

This release bundle ships the public synthetic ExampleCo corridors needed for deterministic replay.

Included runnable fixture roots:

- `minimal/`
- `failed/`
- `stale/`
- `medium/`
- `issued/`
- `cyber-baseline/`
- `ai-governance/`

Additional public-safe support directories such as `exampleco-showcase/`, `lifecycle/`, and `signing/` are included for reference artifacts and release verification.
