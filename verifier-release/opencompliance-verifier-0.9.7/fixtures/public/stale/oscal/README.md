# Stale OSCAL Seed

This directory contains the OSCAL-shaped projection for the synthetic ExampleCo `stale` corridor.

It is generated from checked-in fixture metadata and the machine-readable OpenCompliance control-boundaries file.

Files:

- `opencompliance-stale-catalog.json`
- `exampleco-stale-profile.json`
- `exampleco-stale-ssp.json`
- `exampleco-stale-assessment-plan.json`
- `exampleco-stale-assessment-results.json`
- `family-proxy-targets.json`
- `iso27001-soc2-family-overlap-mapping.json`

Notes:

- This is synthetic OpenCompliance seed content, not framework text.
- The mapping file stays at family-proxy level unless the review pilot says otherwise.
- The corridor stays narrow and scoped to the stale example pack.
