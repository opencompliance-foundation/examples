# Minimal OSCAL Seed

This directory contains the OSCAL-shaped projection for the synthetic ExampleCo `minimal` corridor.

It is generated from checked-in fixture metadata and the machine-readable OpenCompliance control-boundaries file.

Files:

- `opencompliance-minimal-catalog.json`
- `exampleco-minimal-profile.json`
- `exampleco-minimal-ssp.json`
- `exampleco-minimal-assessment-plan.json`
- `exampleco-minimal-assessment-results.json`
- `family-proxy-targets.json`
- `iso27001-soc2-family-overlap-mapping.json`

Notes:

- This is synthetic OpenCompliance seed content, not framework text.
- The mapping file stays at family-proxy level unless the review pilot says otherwise.
- The corridor stays narrow and scoped to the minimal example pack.
